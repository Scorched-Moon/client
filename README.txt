REQUIREMENTS
---------------------
This game requires Python2.5 and Pygame1.8.1 or higher in order to run successfully. The official websites for these are at:
http://www.python.org
http://www.pygame.org

On linux these dependencies can be installed directly from repository on most distro's. On windows and osX these must be downloaded and installed from the internet.
